# Photo Gallery Web App

A React + Vite photo gallery application built as part of a frontend internship pre-screening assignment.

## Features

* Fetches 30 photos from Picsum Photos API
* Displays photos in responsive grid layout
* Real-time search by author name
* Favourite toggle using useReducer
* localStorage persistence for favourites
* Custom hook for API fetching
* useCallback and useMemo for optimization

## Tech Stack

* React
* Vite
* Tailwind CSS

## API Used

https://picsum.photos/v2/list?limit=30

## Project Structure

* reducer/
